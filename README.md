# Marketing_Analytics_Project
End-to-end marketing data analytics project using Excel, SQL, Python, and Power BI to analyze campaign performance and generate business insights.

## Business Problem
The company runs marketing campaigns across multiple channels such as Facebook, Google Ads, Instagram, Email, and YouTube. However, the company is unable to identify:
which marketing channels generate the highest revenue
which campaigns provide the best Return on Investment (ROI)
customer engagement and conversion behaviour
The goal is to analyze marketing data and provide insights to improve campaign effectiveness and marketing strategy.

## Objective
To analyze marketing data and identify key insights related to:
* campaign performance
* customer engagement
* profitability
* Conversion behaviour

## Tools Used
* Excel → Data cleaning, KPI calculation, Pivot Tables
* SQL → Data analysis using queries
* Python (Pandas, Matplotlib) → Exploratory data analysis
* Power BI → Dashboard creation and visualization

## KPIs Analyzed
* Revenue
* Cost
* ROI
* Conversion Rate
* CTR (Click Through Rate)
* Customer Conversions

## Project Workflow
1. Data cleaning and preprocessing in Excel
2. Pivot table analysis in Excel
3. Data analysis using SQL queries
4. Exploratory data analysis using Python
5. Dashboard creation using Power BI

## Key Insights
*Identified top-performing marketing channels
* Compared ROI across campaigns
* Analyzed customer conversion behaviour
* Identified cities generating highest revenue
* Observed revenue trend over time
